# mix-challenge-front
Front-end aplicação confinamento

Aplicação WEB utilizando ReactJs

- Requerimentos:
  NVM - Node Version Manager (Uso para gerenciar várias versões do node instaladas)

- Instalar Node 12.1.0:
  \$ nvm install v12.1.0
  
- Selecionar a versão do node
  -> Na pasta do projeto
  \$ nvm use

- Instalar pacotes
  -> Na pasta do projeto
  \$ yarn install

- Iniciar
  -> Na pasta do projeto
  \$ yarn start

- Parar
  \$ Ctrl + C
